# flip

Prints a random table-flipping emoji to your terminal and copies it to your
clipboard.

## Usage

```bash
./flip.sh
```

## Install

To use it from anywhere, add an alias to your shell config:

```bash
alias flip='/path/to/flip.sh'
```

> macOS only — uses `pbcopy` for clipboard support. Support for other
> clipboards (e.g. `xclip` on Linux) is planned.

## Prerequisites

- macOS
- bash 3.2+
