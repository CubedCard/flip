# flip

Prints a random table-flipping emoji to your terminal and copies it to your
clipboard.

## Usage

```bash
./flip.sh
```

## Install

### Homebrew (recommended)

```bash
brew install cubedcard/flip/flip
```

### Manual

Add an alias to your shell config:

```bash
alias flip='/path/to/flip.sh'
```

## Prerequisites

- macOS, Linux, or Windows (WSL/Git Bash)
- bash 3.2+
